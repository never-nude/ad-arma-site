# Bernini Pieta - Museum Viewer

High-fidelity browser viewer for Bernini's *Pieta*.

This viewer uses the source STL directly with museum-style lighting and mobile-friendly controls.

## Files

- `../../michelangelo/pieta/pieta_source.stl`: Shared source sculpture mesh.
- `index.html`: Interactive Three.js viewer.
